---

## フォルダ構成

| フォルダ | 内容 |
|----------|------|
| **提案資料/** | 顧客向け提案の本文（Markdown）およびスライド用図（drawio） |
| **打合せ・記録/** | 打ち合わせの文字起こし、メール内容などの記録 |

## 主なファイル

- **提案資料/提案資料.md** … 提案の本文（背景・課題と解決策・構成・フェーズ・見積・次のステップ）
- **提案資料/提案資料_デザイン構成.drawio** … スライド用の図（表紙・目次・課題と解決策・構成・フェーズ・見積・次のステップ）
- **打合せ・記録/うち合わせ.md** … 2026年2月6日 株式会社人事部様×サイボウズ kintone 紹介の文字起こし
- **打合せ・記録/メールの内容.md** … サイボウズ営業・岩橋様からのメール（引き合わせの依頼・Y.Fマネジメント様の検討背景）
- **進め方・ワーキング/提案の進め方.md** … 提案方針・スコープ・金額イメージ・体制のメモ
- **進め方・ワーキング/ワーキングフレーム.md** … 作業範囲・成果物・役割・スケジュールの整理（Phase1〜3）
